1 - Hola Mundo en Php, Aproximación

        < ?php include 'includes/header.php'; ?> 
        // El cierre ?> no es obligatorio, recomendable dejarlo abierto.
        //Salvo cuando se combina con Html
    
        < ?php echo "hola Mundo"; ?>

        //El ; siempre es obligatorio para cerrar la sentencia.
        //No es una función como tal, si no una construcción de Lenguaje

        < ?php echo("Hola Mundo");

        print("Hola Mundo");

        print "Hola Mundo";
        //Cuando no lleva () se interpreta como un "constructor de lenguaje"

        print_r("Hola Mundo");
        //print_r para saber el contenido de un arreglo o una variable.

        var_dump("Hola Mundo");
        // var_dump utilizado para debug de cógio, ejemplo para consultar una BD o una variable.

        include 'includes/ footer.php'; ?>

No es recomendable utilizar PHP para generar HTML

   
        Ejemplo:

Extensiones recomendables para Visual Studio Code:

  • PHP Intelephense
  • PHP IntelliSense
  • PHP Debug
  • PHP Serve, similar a live server solo que para Php, click izquierdo, arranca el server

Al utilizar el server también lo podemos ejecutar desde Consola, es decir, desde la carpea en la cual desarrollamos el proyecto...

        php -s localhost:5000  01-hola-mundo.php 
        //Ejemplo del archivo a ejecutar en el servidor, mientras que con Control + c, detenemos el servidor
