14 - For Each

For Each, para recorrer Arreglos

        // For Each
        $clientes = array('Pedro', 'Juan', 'Karen');

        foreach( $clientes as $cliente ): 
            echo $cliente . '< br/>';
        endforeach;

        //El Ciclo for each, genera un alias as $cliente

For

        // For
        for( $i=0; $i < count($clientes); i++){
            echo $clientes[$i] . '< br/>'        //No le pasamos la posición
        }   //as genera un alias $cliente
        endfor;

        // Con el for no le pasamos la extensión, se ejecuta siempre que haya elementos en el arreglo.  
        // count() cuenta o mide la extensión del array
        // sizeof() otra manera

En el caso de recorrer un Arreglo asociativo, el cual parecido al objeto de Js

        $cliente = [

        foreach( $cliente as $key => $valor ):
