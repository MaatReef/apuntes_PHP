8 - Arrays

Los arreglos asociativos en Php, así llamados, son similares a lo que se conoce un objeto en Js

Su indexado, su posición, es igual que en Js

Es importante entenderlo ya que al consultar una Base de Dato, haciendo una consulta SQL, esta viene como un arreglo, por lo cual en PHP tiene una mayor importancia

        $carrito = []; //Más utilizado en Laravel
        $carrito = array(); // Manera más usada en wordpress

        // Cada manera tiene su posibilidad de uso

La etiqueta pre ordena el arreglo, como acá la utilizo para visualizarlo en Html

        < ?php include 'includes/header.php';

        $carrito = ['Tablet', 'Television', 'Computadora'];
                        0          1               2

        // Util para ver los contenidos de un array, var_dump()
        echo "< pre>";
        var_dump($carrito);
        echo "";

La etiqueta < pre > más el var_dump() para una mejor visualización

        // Acceder a un elemento del array
        < pre>
        var_dump($carrito);
        < /pre>
        echo $carrito[1]; // Accedemos a la posición Televisión

        // Añadir un elemento nuevo al final, sin conocer la extensión, ni posición del array.
