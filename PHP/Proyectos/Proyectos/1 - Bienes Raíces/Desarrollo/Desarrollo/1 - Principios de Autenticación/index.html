1. Principios de Autenticación

Agregamos credenciales de inicio de sesiónAsí evitaremos que cualquier persona pueda ingresar al panel de Administración.
Buenas prácticas para autenticar usuarios..
Principios básicos de autenticación.

        Hay diferentes formas de autenticar usuarios:
        - Con algo que usuario recuerda (password)
        - Con un dispositivo físico (Un mensaje a un celular)
        - Con algo físico (Huella digital o Cara)

Algoritmo para autenticar... Email y password deben coincidir.

        Otras validaciones incluyen:
        -Si el usuario existe
        -Si el usuario existe pero el password es incorrecto
        -Si ambos son correctos entonces autenticar el usuario

PHP y Autenticación:

Php contiene funciones para hashear passwords (no es lo mismo que encriptar)..
También hay una función para verificar un password.
Un password hasheado no se puede revertir.
Si el usuario se le olvida deberá crear un nuevo password.
