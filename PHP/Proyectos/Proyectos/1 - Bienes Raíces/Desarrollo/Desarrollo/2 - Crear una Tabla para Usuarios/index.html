2. Crear una tabla para usuarios

En este proyecto no tendremos usuarios aleatorios que podrán registrarse, solo usuarios previamente definidos por la propia empresa.
En el proximo proyecto, si se podrá crear usuarios aleatorios.

En este proyecto autenticaremos un unico usuario, que sera el administrador.

EN la raíz del proyecto creamos un archivo llamado usuario.php..
Al subir el proyecto a producción, es necesario eliminar dicho archivo.

El siguiente es un Snippet de código.. Útil

        < ?php

            // Importar la conexión 1ª			//Pasos
            require 'includes/config/database.php';
            $db = conectarDB();

            // Crear un email y password, autenticado con email como facebook o twitter
            $email = "correo@correo.com";
            $password = "123456";

            $passwordHash = password_hash($password, PASSWORD_BCRYPT);


            // Query para crear el usuario
            $query = " INSERT INTO usuarios (email, password) VALUES ( '${email}', '${passwordHash}'); ";

            // echo $query; //Solo para observar la variable email y passwordHash

            // Agregarlo a la base de datos
            mysqli_query($db, $query);

Creamos una tabla en la base de datos llamada "usuarios", junto con las otras dos de propiedades y vendedores.
Dicha tabla tendra un id de tipo int(1) auto incrementable, un email de tipo varchar(50) y un password de tipo char(60).
