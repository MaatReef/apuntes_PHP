17 - Require

        < ?php include 'includes/header.php';
        // No colocarlo entre (), recomendación


        require 'funciones.php'; //Llamamos al archivo funciones.php que esta en la misma carpeta.
        // Require; para cuando usamos funciones críticas para la app, ejemplo conectar a una BD.
        
        iniciarApp();

        < ?php require_once 'includes/header.php';
        // require_once; lo incluye una vez, si ya fue incluido lo ignora 
        
        include 'includes/footer.php';

Recomendación: Separar las funciones o archivos necesarios, para una mejor organización, de esta manera las incluimos o llamamos desde otros archivos en el cual se requiera
